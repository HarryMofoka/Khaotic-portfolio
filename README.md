## ‚ú® Key Features

### 1. "Enter the Mind" (Hero)
A generative noise field powered by Simplex Noise and HTML5 Canvas. The background reacts to cursor movement, creating a unique texture for every visitor.

### 2. "Projects That Breathe"
Portfolio pieces rendered as film-stills with:
- **Video Previews**: Instant feedback on hover.
- **Micro-Rotations**: Randomized angles for a scattered layout.
- **Parallax Depth**: Floating layers that shift with scroll.

### 3. Sensory Polish
- **Context-Aware Cursor**: A custom cursor system that changes state based on element interaction (Default, View, Lab, Text).
- **Ink Trail**: A canvas-rendered fading trail that follows the mouse.
- **Multi-Mood System**: Cycle between *Default (Dark)*, *Light*, *Midnight (Blue)*, and *Ember (Amber)* themes.
- **Procedural Audio**: An opt-in ambient drone system generated in real-time.

### 4. "The Lab" & "The Creator"
- **The Lab**: A freeform grid showcasing experimental work with status-aware badges.
- **The Creator**: A personalized about section featuring an infinite-scrolling skills marquee and GSAP-revealed portrait.
